# Service Design × Business Architecture × Journey Operations

A comprehensive knowledge base exploring the convergence of Service Design, Business Architecture, Enterprise Architecture, and Journey Operations — with focus on large retail organizations.

## 📚 Overview

This repository contains research, synthesis, and practical guidance for transformation practitioners and senior leadership navigating the intersection of:

- **Design Thinking & Service Design** — Human-centered approaches to experience design
- **Business & Enterprise Architecture** — Capability-based planning and strategic alignment
- **Journey Operations** — Operationalizing customer journey management at scale

## 🎯 Who This Is For

- **Senior Leadership** seeking strategic understanding of CX-Architecture convergence
- **Transformation Practitioners** implementing journey-centric operating models
- **Service Designers** connecting design work to enterprise architecture
- **Enterprise Architects** integrating customer experience into architecture practice
- **Journey Managers** establishing governance and measurement frameworks

## 🔧 Generate PDF Locally

The HTML file is designed for browser print-to-PDF:

1. Open `SYNTHESIS.html` in any browser
2. Press `Ctrl+P` (Windows/Linux) or `Cmd+P` (Mac)
3. Select "Save as PDF"
4. Use A4 format with default margins

Or use command-line tools:

```bash
# Using wkhtmltopdf
wkhtmltopdf SYNTHESIS.html SYNTHESIS.pdf

# Using pandoc + LaTeX
pandoc SYNTHESIS.md -o SYNTHESIS.pdf --pdf-engine=xelatex

# Using Chrome headless
google-chrome --headless --print-to-pdf=SYNTHESIS.pdf SYNTHESIS.html
```
